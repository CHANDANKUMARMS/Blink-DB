\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename K, typename V$>$\newline
class LRUCache$<$ K, V $>$}
A templated Least Recently Used (LRU) Cache. 

This class implements an LRU cache using a combination of an unordered\+\_\+map and a doubly-\/linked list. The cache evicts the least recently used item when its capacity is reached.
