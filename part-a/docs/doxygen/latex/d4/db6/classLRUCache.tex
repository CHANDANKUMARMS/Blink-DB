\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename K, typename V$>$\newline
class LRUCache$<$ K, V $>$}
A template class for a Least Recently Used (LRU) cache. 

This class implements a cache that evicts the least recently used item when the capacity is reached. It supports basic operations like getting, putting, and removing items.
