\doxysubsection{Detailed Description}
Header file for the \doxylink{classBlinkDB}{Blink\+DB} key-\/value storage engine. 

This file declares the \doxylink{classBlinkDB}{Blink\+DB} class which implements a simple key-\/value storage engine with in-\/memory memtable and SSTable persistence, along with an LRU cache for fast GET operations.
