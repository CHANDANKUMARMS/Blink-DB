\doxysubsection{Detailed Description}
A simple key-\/value storage engine. 

The \doxylink{classBlinkDB}{Blink\+DB} class provides functionality for setting, getting, and deleting key-\/value pairs. It uses an in-\/memory memtable and persistent SSTable files. It also leverages an LRU cache for fast retrieval of frequently accessed keys.
